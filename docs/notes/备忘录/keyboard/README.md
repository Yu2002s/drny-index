一些开发工具的快捷键

## Jetbrains

```
shift+n 任意位置换行
Ctrl + Shift + I Open quick definition lookup 打开定义快速查找
Ctrl+E 最近打开的文件
ctrl+Shift+N 通过文件名快速查找工程内的文件（必记）
ctrl+Shift+alt+N 通过一个字符快速查找位置（必记）
alt+7 显示类结构
alt+left/right 快速切换tab标签
alt+up/down 在方法间快速移动光标
ctrl+w 选择单词
ctrl+f 在当前文件进行关键字搜索
ctrl+0 快速重写方法
ctrl+h 显示类的层次结构（继承关系）
ctrl+g 快速定位到行列
ctrl+[/] 定位到代码块开始/结束位置
ctrl+n 快速搜索类和文件
ctrl+u 快速转到类的父类
ctrl+alt+t 将选中代码块使用if/try/catch包裹起来
ctrl+alt+i 自动缩进行
ctrl+p显示方法参数信息
ctrl+shift+space 自动补全代码
ctrl+i 实现方法
```
